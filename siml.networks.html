siml.networks.network.add_block(name, block, *, trainable=True, use_support=False)¶

Add block definition to siml.

Parameters

  • name (str) – Name of the block.

  • block (siml.network.SimlModule-like) – User defined block.

  • trainable (bool, optional) – If True, the block is considered as a trainable block. The default is True.

  • use_support (bool, optional) – If True, use sparse matrix for the second input. The default is False.

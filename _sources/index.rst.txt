SiML: A Simulation ML library
=============================
*SiML* is a machine learning library focusing on geometrical and simulation
data.
It facilitates machine learning processes, including preprocessing, learning,
and prediction.
In particular, SiML has handy functionalities to handle geometrical data,
which are in graph or point cloud formats in general.


Installation
============

SiML supports Python 3.9 or newer.
SiML can be installed via pip
(note that the name of the package is :code:`pysiml`):

.. code-block:: bash

  $ pip install pysiml

However, the name of the module used to import is :code:`siml`.

.. code-block:: python

  import siml
